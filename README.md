# PCBAssemblyConverter

基板設計ソフトから出力された BOM / P0S ファイルを、各基板製造業者の入稿ファイルに変換する Web App です。  
変換された BOM / POS ファイルと、変換ルールをまとめた JSON ファイルが ZIP でダウンロードできます。

## アプリケーション URL

https://koktoh.github.io/PCBAssemblyConverter/
